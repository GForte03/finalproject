Accessibility

Designing so more people can use the site (keyboard, screen readers, contrast, and structure).

Accessibility practices I used

  • Semantic structure (header, nav, main, section, footer)
  • A “Skip to main content” link for keyboard users
  • Visible focus styles (:focus-visible)
  • Alt text for images (especially screenshots)
  • Clear headings in order (H1 → H2 → H3)
  • Descriptive link text (not “click here”)

Example: accessible form label

Labels should be connected to inputs using for and id. 

<label for="email">Email</label>
<input id="email" type="email" />
