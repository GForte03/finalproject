CSS styling and layout

Using consistent styles, modern layouts, and responsive patterns.

Styling basics

I can alter color, font, spacing, and focus/hover states with CSS. I even utilized CSS variables for this website to maintain this pink theme throughout all of the pages. 

:root {{
  --accent: #d63384;
  --border: #f2b7d8;
}}

Layout with Flexbox and Grid

For one-dimensional layouts (rows or columns), I use Flexbox, and for two-dimensional layouts (rows and columns), I use Grid. Grid is used for the main content and sidebar in this project, while Flexbox is used for the overall layout. 

/* Flexbox: page layout */
.site {{
  display: flex;
}}

/* Grid: content + table of contents */
.grid {{
  display: grid;
  grid-template-columns: 1fr 280px;
}}

Responsive design

The website functions better on PCs, tablets, and phones thanks to responsive design. In order to have the left navigation appear at the top on smaller displays, I utilized media queries. 

@media (max-width: 720px) {{
  .site {{ flex-direction: column; }}
  .sidebar {{ width: auto; height: auto; }}
}}
