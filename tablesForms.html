HTML tables and forms

Using tables for data and forms for user input—accessibly.

Tables

Tables are best for tabular data (information that belongs in rows and columns). Using <caption>, <th>, and proper headers makes the table more accessible.

Forms

Forms collect user input. I try to use clear <label> text, logical grouping with <fieldset>, and helpful placeholders (without relying on placeholders as labels).
