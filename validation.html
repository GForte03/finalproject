Code validation

Using W3C validators to catch errors in HTML and CSS.

Why validate?

Errors such as CSS typos, improper nesting, and missing closing tags can be detected using validation. Additionally, it simplifies debugging and increases cross-browser stability.

How I validate HTML and CSS

  1. Copy/paste my HTML into the W3C HTML Validator.
  2. Fix errors (not just warnings) and re-check.
  3. Paste my CSS into the W3C CSS Validator and fix issues.
